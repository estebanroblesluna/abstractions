  Web development
  We use precise software engineering methods to develop software using the ubiquity of the Internet to integrate applications and strategic information, both in traditional areas and in developments for mobile equipment, e-commerce and e-learning.
  Mobile & distributed systems
  The development of solutions for distributed systems requires a complete architectural vision in order to guarantee qualities such as high performance, robustness and flexibility, within a technological environment which continously evolves. The inclusion of mobile devices imposes new requirements to distributed systems developed in N-Tiers since the physical location of the user must be considered.
  Groupware systems
  Groupware applications let a group of users communicate and work with each other in collaborative manners. This usually implies that the group be able to organize activities, solve problems, edit documents, perform negotiations, and so on. We use specific technologies based on electronic services to send email, support video conferences, chats and others ways of interaction.
   
Geographic informacion systems
Geographic information systems are used to collect, analize and later present information describing logical and physical properties of real_world entities. Today, different sciences are using this techonology, which adds an extra level of complexity when designing flexible GIS system to cover a minimal set of functionalities.
