References

LIFIA offers its services and capabilities in different scenarios and markets. We present a short list of the most representatives projects we have developed in cooperation with public and private institutions.

Department stores

Emerson Process Management (2001-2005)
We have developed a Web & Desktop platform which expedite the internal management and facilitate the monitoring of the shopping orders as well as the booking of the products on behalf of the people related to the company. Main techonologies applied were: JSP/Struts; Delphi and the MSSQL Database Server.

Almacenes Paris (2003 - ongoing)
We created a group of management oriented applications to administer large department stores and/or warehouses. These applications include stock administration and control.

Healthcare Administration

Medife (2000 - ongoing)
Internet based management system that covers all the administrative workflows present in a private healthcare institution. All related activities are currently supported (providers, beneficiaries, assistance, invoicing and payment). Main technologies we are using are Oracle 8i Database; JSP; and Delphi.

Games / Casinos

Casino ( 2002 – ongoing )
The Casino team develops games and technology for ID Interactive, a gaming company based in Florida. Based on a concept and artwork from a creative department, the final product is a cabinet containing all the hardware and software neeeded to run a game in a casino.

Public Administration

Secretaría de Hacienda del Ministerio de Economía y Producción de la Nación (2004 - ongoing)
System for drawing up and administrating the national budget. The proyect covers the definition of the basic architectural components, development of eSidif system and paralelly the training in object technology and J2EE for the members of the Computer Science Unit.

HARV - Vehicle routing and position adjustment tool (2005)
Development of GIS tools for digital cartography manipulation(dgn, ShapeFile (R), cgm). Creation of routing algorithms to adjust a network of street positions (represented in a digital cartography). We used .NET frameworks, Geomedia products and a MSSQL Database Server.

University Administration

Facultad de Agronomía de la UBA (1999 - ongoing)
Internet based software to manage all the administrative tasks related with students, courses and exams of a public faculty of the Buenos Aires University. This application features a statistical module that estimates the number of student for each forthcoming course.

Telecomunications

Impsat (2002 - ongoing)
Intranet development for a telecomunications company based on several countries (Argentina, Peru, Brazil and USA). The application manages all the business entities, such as carriers, customers, paths, etc. We used a J2EE 3 tier architecture connected to an Oracle Database.

Banks and financial applications

JP-Morgan (1990-2001)
Development of financial applications using object oriented technologies: decision support systems, securities and stock position calculation; activity reports, etc.

Banco Galicia (2003-2004)
Architectural definition and conceptual design of the module named "Customer Card". Object oriented design mentoring. We have also dictated several training courses about object orientation and Use Cases.

Digital Moderation Tools (2004)
Development of digital moderation components to be integrated in the Digimod Project of Fraunhofer IPSI (Germany). The moderation tools were developed on top of a Java framework for digital moderation developed by IPSI. Development took place in our installations in La Plata, with continuous contact with the framework developers in Germany.
