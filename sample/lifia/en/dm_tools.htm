Digital Moderation Tools  
Area
Solutions development
Sub Area / Vertical Markets
Groupware systems
Execution time frame
April - June 2004
Customer
Fraunhofer IPSI. Darmstadt. Germany
Project leader
Mg. Javier Bazzocco
Project´s Brief
Development of digital moderation components to be integrated in the Digimod Project of Fraunhofer IPSI.
Project´s scope
DIGIMOD is a project hosted at the Fraunhofer IPSI (Germany) research and development laboratory. Its main objective is developing tools for digital moderation of meetings. The process is as follows: a person acting as moderator uses a set of components to simulate some activities that are usually performed by hand; each participant interacts through a computer with specific components selected for that session (e.g: idea-gathering component). The moderator is permanently in control of the session by means of another administration component.
LIFIA participated in this project by designing and developing two digital moderation components: a brainstorming based ideas gathering component and an ideas ranking component (which is based in the previouos component).
The development was based on a framework created at IPSI, which has the following important features: support for components intercommunication; concurrence control; persistence and failure recovery.
Each component has two different interaction modes (or views): one for the participants to interact with it when active and the other corresponding to the administrator or mediator that permits only the interaction of the moderator while other participants just can observe.
Technologies and methodologies we used
We developed the components using Java (version 1.4.2). We chose XML as the preferred way to exchange information between the server and its clients. Our develpment environment included Eclipse 3.0, ANT, CVS (as source code repository), Mantis (bug tracking) and JUnit to run the unit tests.
Finally, as an example of how we can adapt to the Customer´s standards we applied codification rules proposed by IPSI and reached 80% of code coverage (Junit).
