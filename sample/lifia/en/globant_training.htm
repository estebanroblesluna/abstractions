Training course Globant S.A. (2006)

 
Area
Corporate Training
Period
July - August (2006)
Customer
Globant - http://www.globant.com/en/index.html
Project leader
Dr. Alejandro Fernández
Project´s brief
Training course on Object technology and basic Java for the new members and employees from different areas of Globant’s staff
Objectives
Provide the assistants with the essential concepts related to design base don the Object-Oriented and Java programming paradigm.
Project´s scope

Scope of the project
Total number of trainees (already trained and to be trained) 35
Topics and technology
• Introduction to Objects
• Object-oriented design
• Basic Java
• Basic UML
• AdvancedJava
• Design Patterns
Current status
The training course for the above mentioned assistants has been successfully completed. Additional iterations are already planned for the next months.
