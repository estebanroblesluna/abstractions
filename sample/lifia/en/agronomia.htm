Web development - University Administration

Execution time frame
2000 - ongoing
Customer
Agronomy School of University of Buenos Aires
Project leader
Lic. Luis Mariano Bibbó
Project´s brief
Enrollment system for final exams and regular courses via internet and administrative management of the student's department. The system also has a statistical module for professors.
Objectives
We developed an information system to support all circuits related to the enrollment in final exams and regular courses via internet. All administrative tasks are also supported. Proffesors have a statistical module to organize in a better way, incomming enrollments to their future courses.
Project´s scope
The project´s scope included two major elements: the enrollment information system and a file tracking system to help improve all the administrative circuits.
Technologies we use
• Oracle database
• IAS Application server
• Java language
Current status
We are currently performing maintenance activities for both systems.
