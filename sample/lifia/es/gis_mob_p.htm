SIG y computación móvil

Nuestro objetivo es investigar el problema de construcción de aplicaciones geográficas y en particular aplicaciones móviles desde la perspectiva de la ingeniería de software, investigando acerca de enfoques arquitecturales, métodos y modelos de desarrollo poniendo especial énfasis en aspectos de calidad de dichas aplicaciones.

Transferencia de los resultados de investigación
El objetivo del área es la utilizacion metódica de técnicas para la construcción de aplicaciones geográficas y manipulación de datos espaciales. En este momento se está realizando un trabajo sobre ajuste de posiciones provenientes de GPS a ejes viales. Se está importando cartografía DGN y shapeFile a SQL Server y se está generando DGN a partir de datos almacenados en SQL Server. Se está generando cartografia para la Web en formato CGM desde la base de datos SQL.
