Referencias

El LIFIA realiza la prestación de sus servicios y la aplicación de sus competencias en diversos ámbitos y diferentes mercados. A continuación se detallan algunos de los casos más representativos.

Grandes tiendas

Emerson Process Management (2001-2005)
Desarrollo de sistemas Web y Desktop que agilizan la gestión interna y facilitan el seguimiento de órdenes de compra y reservas de productos por parte de personas relacionadas con la empresa. Las tecnologías empleadas para el proyecto son JSP-Struts-Delphi y el motor de base de datos es MSSQL Server.

Almacenes Paris (2003 - continúa)
Conjunto de aplicaciones para administración de grandes tiendas y/o depósitos. Control y publicación en Internet de Stock y seguimiento de operaciones.

Administración de Salud

Medife (2000 - continúa)
Desarrollo de un sistema de gestión integral con arquitectura Intranet (portal) para empresa de servicios de salud: circuitos y procedimientos, diseño y desarrollo del sistema de gestión de la empresa - beneficiarios, prestaciones, facturación y liquidación, migración, depuración de base de datos e implementación.

Juegos / Casinos

Casino ( 2002 – continúa )
Desarrollo de software embebido en máquinas de juegos de azar para casinos. El desarrollo abarcó todas las capas del dominio de los juegos de azar, desde el desarrollo de las librerías multimedia de alta performance hasta la administración y coordinación de varias máquinas de juegos en un servidor. El desarrollo respeta los estándares establecidos para los casinos de EEUU y Brasil.

Administración Pública

Secretaría de Hacienda del Ministerio de Economía y Producción de la Nación (2004 - continúa)
Sistema para formulación y administración del presupuesto del Gobierno Nacional. El proyecto contempla la definición de los componentes básicos de la arquitectura, construcción del sistema eSidif y paralelamente la capacitación en tecnología de Objetos y J2EE para los miembros de la Unidad de Informática.

HARV - Herramienta de Ajuste y Ruteo Vehicular (2005)
Desarrollo GIS de herramientas de manipulación de cartografía digital (dgn, ShapeFile (R), cgm), y generación de algoritmos de ajuste de posiciones a una red de calles representada mediante cartografía digital. El proyecto empleó los frameworks de .NET, Geomedia y Geomedia Web Map utilizando como motor de base de datos MSSQL Server.

Administración Universitaria

Facultad de Agronomía de la UBA (1999 - continúa)
Sistema de inscripción de alumnos a finales y materias vía Internet y manejo administrativo del departamento de alumnos y de seguimientos de expedientes. El sistema cuenta además con un módulo estadístico para los docentes.

Telecomunicaciones

Impsat (2002 - continúa)
Desarrollo de una Intranet para gestionar el negocio de una empresa de telefonía con sedes en diferentes países (Argentina, Perú, Brasil, EEUU). Administra todas las entidades del negocio, como carriers, clientes, rutas, numeración, destino, etc. El desarrollo consiste de una aplicación Web con tecnología J2EE con una arquitectura de 3 capas vinculada con una base de datos Oracle.

Bancos y aplicaciones financieras

JP-Morgan (1990-2001)
Desarrollo de aplicaciones financieras utilizando tecnologías orientadas a objetos: captura de operaciones y cálculo de posiciones de securities y acciones, posiciones consolidadas, informes de actividad y toma de decisión.

Banco Galicia (2003-2004)
Definición de arquitectura y diseño conceptual para el módulo de soporte de Ficha de Cliente. Servicio de consultoría para diseño orientado a objetos. Además se han dictado en varias oportunidades diferentes cursos de capacitación sobre tecnología de objetos, y casos de uso.

Digital Moderation Tools (2004)
Desarrollo de componentes para la moderación digital de reuniones. Estos componentes se integran en el marco del proyecto Digimod del Instituto IPSI de Fraunhofer (Alemania). Estos componentes fueron implementadas sobre un framework Java desarrollado para la moderación digital por el IPSI. El proyecto tuvo lugar en nuestras oficinas en La Plata, manteniendo contacto continuo con los desarrolladores del framework en Alemania.
