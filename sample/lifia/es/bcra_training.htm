Capacitación BCRA  
Área
Capacitación corporativa
Sub área / Mercado vertical
Administración pública
Período de realización
Junio 2006 – julio 2007
Destinatario y/o Cliente
Banco Central de la Nación
Director/es del proyecto
Dr. Alejandro Fernández
Resumen del proyecto
Capacitación en tecnología de objetos y plataformas de desarrollo JEE y .NET para integrantes del plantel de desarrolladores del Banco Central.
Objetivos
Capacitar a los asistentes en los conceptos mínimos y esenciales relacionados con el diseño basado en el Paradigma de Orientación a Objetos y en tecnologías .NET y JEE.
Alcance del proyecto
Dimensión de la audiencia capacitada/a capacitar: 15 personas

Temas y tecnologías:

• Introducción a Objetos
• UML Básico
• Diseño con Objetos
• .NET Basico
• Taller .NET. Visual Studio, Source Safe
• Persistencia con .NET
• Taller de Persistencia con .NET
• Desarrollo de Aplicaciones Web con ASP.NET
• Taller de Desarrollo de Aplicaciones Web con ASP.NET
• Java Básico
• Taller de Java Básico
• Persistencia con Hibernate
• Taller de Persistencia con Hibernate
• Desarrollo de Aplicaciones Web con JEE
• Taller de Desarrollo de Aplicaciones Web con JEE
• Desarrollo Avanzado de Componentes de Negocio con Spring
• Taller de Desarrollo Avanzado de Componentes de Negocio con Spring
  Tecnologías utilizadas
  JEE, .NET
  Situación actual
  En la actualidad se desarrollan los cursos y talleres de la plataforma .NET. Para La capacitación en la plataforma JEE está planificada para principios de 2007.
