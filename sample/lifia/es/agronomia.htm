Agronomía, Sistema de Alumnos y manejo de expedientes

 
Área
Desarrollo de soluciones
Sub área / Mercado vertical

Desarrollo web - Administración universitaria

Período de realización
2000 - continúa
Destinatario y/o Cliente
Facultad de Agronomía de la Universidad de Buenos Aires
Director/es del proyecto
Lic. Luis Mariano Bibbó
Resumen del proyecto
Sistema de inscripción de alumnos a finales y materias vía Internet y manejo administrativo del departamento de alumnos y de seguimientos de expedientes. El sistema cuenta con un módulo estadístico para los docentes.
Objetivos
El proyecto consistió en desarrollar una aplicación web que recibe una fuerte carga en los días de inscripción. Los alumnos de la facultad deben inscribirse a los exámenes de las materias, a los horarios y a sus cursos. Los docentes cuentan con un módulo estadístico que permite prever la cantidad de inscriptos que tendrán sus materias para lograr una mejor distribución de los horarios.
Alcance del proyecto
Este proyecto incluyó dos grandes componentes: el sistema de información para inscripciones y un sistema de seguimiento de expedientes que ayuda a mejorar todos los circuitos administrativos.
Tecnologías utilizadas
• Base de datos Oracle
• Servidor de aplicaciones IAS
• Lenguaje Java
Situación actual
Actualmente estamos llevando a cabo tareas de mantenimiento y extensión en ambos sistemas.
