  JavaScript by example
  (por Leandro Yalet) - 17/Julio/2008 - Permalink
 

    Como habíamos prometido esta nota va a ser más corta y vamos a ver ejemplos concretos utilizando una biblioteca de JavaScript para mostrar el poco esfuerzo que requiere hacer cosas interesantes hoy en día.

    Específicamente vamos a usar Mootools ya que en un principio fue basada en Prototype así que el aprendizaje en nuevas bibliotecas similares no sería mayor esfuerzo.

    Como noticia el 12 de Junio fue el lanzamiento oficial de Mootools 1.2, nació con un peso de 90KB (sin compresión), también se puede optar comprimir el archivo (60KB usando el YUI compresor, y alrededor de 18KB si se usa gzip!!! En otra nota comentaremos como).

Menús con la ayuda de Mootools
Antes de mirar los ejemplos primero hay que entender algunas cosas para poder indagar un poco el funcionamiento de estos menúes, para ello aquí se presenta un breve resumen de selectores de CSS:

$('myElement') //obtiene el elemento con el id "myElement"
$$('a') //obtiene un array con los elementos a (anchor) del documento.
$$('a', 'b') //obtiene un array con los elementos anchor y elementos b (bold) contenidos en el documento.
$$('#myElement a.myClass') //un array con elementos anchor de clase "myClass" contenidos en el elemento con id "myElement"
$E('a', 'myElement') //retorna el primer anchor dentro del elemento con el id "myElement"

    De esta manera los elementos que han sido recolectados son extendidos, es decir, adquieren el comportamiento de una clase definida en la biblioteca llamada Element (http://docs111.mootools.net/Native/Element.js).

    Ya con estas ideas podemos ver los ejemplos, como aclaración estos menús no son parte de la biblioteca, es necesario agregar el ".js" correspondiente y además agregar las hojas de estilos indicadas. Naveguen los links para verlos en acción.

JS
var myMenu = new ImageMenu($$('#imageMenu a'), {
openWidth:310,
onOpen:function(e,i){console.log(e)}
});
     Como podemos evidenciar donde se indica "$$('#imageMenu a')" no es más que un array con los anchor dentro del elemento div con id "imageMenu".

Fancy Menu
http://devthought.com/cssjavascript-true-power-fancy-menu/

HTML
<div id="fancymenu">
<ul>
    <li class="current" id="menu_home">
        <a href="#">Home</a>
     </li>
    <li id="menu_plantatree">
        <a href="#">Plant a tree</a>
    </li>
    <li id="menu_travel">
        <a href="#">Travel</a>
     </li>
    <li id="menu_rideanelephant">
        <a href="#">Ride an elephant</a>
    </li>
</ul>
</div>

JS
window.addEvent('domready', function() {
new SlideList($E('ul', 'fancymenu'), {
transition: Fx.Transitions.backOut, duration: 700, onClick: function(ev, item) { ev.stop(); }});
});

    Similar al ejemplo anterior, pero aquí se recolecta solo el elemento ul dentro del elemento div con id "fancymenu". La limitación de este menú es que es restrictivo al uso de listas con elementos li (inspeccionen su implementación).

    La ventaja de estos menús es que a pesar de que el browser no utilice JavaScript, o el usuario lo deshabilite, la funcionalidad del menú no se ve alterada y es posible navegar los links. Gran parte de su atractivo, por no decir el 100%, está en los estilos definidos en los CSS, estos menús son ejemplo de que JavaScript sirve para enriquecer la apariencia de una página web y no atarla a un requerimiento tecnológico.
