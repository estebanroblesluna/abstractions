  Sistemas distribuidos y móviles  
El desarrollo de soluciones para sistemas distribuidos requiere una visión arquitectural completa para garantizar cualidades como alto rendimiento, robustez y flexibilidad, dentro de un marco tecnológico que evoluciona constantemente. La incorporación de dispositivos móviles introduce nuevos requerimientos para los sistemas distribuidos desarrollados en N capas ya que ahora la ubicación física del usuario juega un papel preponderante.
Experiencia del LIFIA
La variedad de proyectos ejecutados existosamente (tanto nacional como internacionalmente) nos dan una vasta experiencia en este campo, permitiendonos desarrollar proyectos complejos relacionados con la definición e implementación de sistemas distribuidos y/o móviles.
Servicios ofrecidos por el LIFIA
Desarrollo de soluciones:
El LIFIA es capaz de desarrollar proyectos completos relacionados con sistemas distribuidos, desde las etapas tempranas de especificación de requerimientos hasta las etapas de implementación, posterior puesta en producción, utilizando las últimas tecnologías adoptadas por el mercado. Los desarrollos pueden realizarse en casa del Cliente o en nuestra oficinas.
Capacitación:
El LIFIA cuenta con profesionales altamente capacitados que dictan cursos de capacitación en casa de Cliente o en nuestras oficinas sobre productos o metodologías de trabajo relacionados con sistemas distribuidos y desarrollo para dispositivos móviles.
El calendario usualmente se establece de común acuerdo entre el LIFIA y el Cliente.
Productos and estándares
• Servidores de bases de datos Oracle, MySql & Sql Server
• Estándares J2EE & J2ME, .Net
• Mapeador Objeto / Relacional Hibernate
• Framework de middleware Spring
• Frameworks web Struts & Tapestry
• Servidores de aplicaciones IBM Websphere, BEA Weblogic, JBoss, Jakarta Tomcat
